模块的用途

多重继承 
module A
  module People
    def say
      puts "hello"
    end
  end
  module Easter
    def eat
      puts 'eat'
    end
  end
end

class Chinese
  include A::People
  include A::Easter
end

p = Chinese.new
p.say # hello
p.eat # eat

block与声明式编程

ruby中的循环

原始形式(外部迭代器) 
iter = 2.upto(5)
while iter.next
  p 'hello'
end
改为block(内部迭代器) 
2.upto(5) do
  p 'hello'
end
支持参数 
2.upto(5) do |i|
  p 'hello'+i
end

block的作用

改进groovy的例子 
(0..10).each{
  i -> pool.submit( { println i } as Runnable );
}
ruby的例子 
(0..10).each{
  |i| pool.submit { print i };
}
也可以写成 
(0..10).each do
  |i| pool.submit { print i };
end
总结：block使得变量局部化

语言与设计模式

  • 先来看一个method missing的例子

class MyFoo
    def method_missing(method, *args, &block)
        puts %Q[method: #{method}
args: #{args.inspect}
on: #{self.inspect}
]
    end
end

ruby版代理模式
class MyProxy
    def initialize(real_account)
        @subject = real_account
    end

    def method_missing(name, *args)
        puts("Delegating #{name} message to subject.")
        @subject.send(name, *args)
    end
end
