Mar 4th, 2012 | Comments

git remote prune

说明：remote上的一个分支被其他人删除后，需要更新本地的分支列表
使用举例：

git push -f origin master

说明：强制push本地代码，例如在本地进行 amend 模式的commit的时候，实际上是重写了原来的commit，如果直接push会出错，此时需要强制push，让remote端与本地一致
