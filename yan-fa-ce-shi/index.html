When /(.+)开始(.+)/ do |user, behaviour|
  user = find_or_create_user(user)
  case behaviour
  when '管理自己的指令'
    @browser.link(:text, '管理我的指令').click
  end
  @browser.link(:text, behaviour).click
end

小结：一致性通常能提高复用水平

agent技术选择

  • 面向网络编程

  • 有一个支持异步调用的ssh库

  • 能够简化并发编程

  • 有针对网络和分布式编程的基础设施

  • 可靠性好

  • 方便进行水平拓展（可选）

erlang

center技术选择

  • 面向web开发的基础设施

  • 允许业务模型多变（简单易用的ORM）

  • 支持快速原型

  • 原型可用，可迭代

  • 学习成本有限（如果不是现有技术）
